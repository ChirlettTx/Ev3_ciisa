Descripción del Proyecto

Este proyecto se centra en la implementación de un servidor Rocky Linux, abarcando una serie de actividades técnicas esenciales para su correcta configuración y administración. Las principales áreas de trabajo incluyen:

  1. Administración de Paquetes y Repositorios YUM (Indicador 7.1): Gestión y actualización de paquetes mediante el gestor de paquetes YUM, asegurando la correcta instalación de las herramientas necesarias.
  2. Identificación y Uso de Herramientas de Monitoreo (Indicadores 5.1, 5.2, 5.3): Selección e implementación de herramientas para monitorear el rendimiento del servidor y garantizar su estabilidad operativa.
  3. Interpretación de Resultados y Medidas Correctivas (Indicadores 6.1, 6.2, 6.3): Análisis de los resultados obtenidos de las herramientas de monitoreo, con el fin de identificar posibles fallos y aplicar medidas correctivas adecuadas.
  4. Instalación y Configuración del Servidor Web Apache (Indicador 7.2): Implementación del servidor web Apache para facilitar la gestión y visualización de contenidos web.
  5. Implementación de un Firewall Básico (Indicador 7.3): Configuración de medidas de seguridad básicas para proteger el servidor de accesos no autorizados.

Además de los aspectos técnicos mencionados, el proyecto incluye la creación de una página web utilizando HTML y CSS. Para garantizar la organización y el progreso eficiente del trabajo, se emplea la gestión de proyectos a través de la plataforma Jira, lo que facilita la coordinación y el seguimiento de tareas. Las secciones de respuestas proporcionan un conocimiento profundo sobre el uso de herramientas basadas en inteligencia artificial (IA), destacando la importancia de su aplicación ética y efectiva en el entorno laboral.
